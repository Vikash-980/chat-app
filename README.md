## ✨ Features

* **Real-time Messaging:** Messages are delivered instantly without the need for page reloads.
* **Socket.io Integration:** Utilizes WebSocket technology for seamless, bi-directional communication.
* **Responsive UI:** Fully optimized for a smooth experience across Mobile, Tablet, and Desktop devices.
* **Clean & Minimal Design:** A user-friendly interface focused on simplicity and ease of use.

---

## 🛠️ Tech Stack

| Component | Technology Used |
| :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript (React optional) |
| **Backend** | Node.js, Express.js |
| **Real-time Engine** | Socket.io |
| **Hosting** | Render |

## 🚀 Local Setup

To run this project locally on your machine, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/Vikash-980/chat-app.git](https://github.com/Vikash-980/chat-app.git)
   cd chat-app
   npm install
   PORT=3000

   ```
